# flask-app

This is a Basic Flask - React Application. 

**#installation**

- Clone the project. 
**Frontend**
- install node modules in react folder (flask-app/react/)
- npm start
**Backend**
- install gunicorn and run app on localhost:8000

Open in browser http://localhost:3000
